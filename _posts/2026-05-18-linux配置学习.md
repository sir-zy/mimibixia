今天首次在 Linux 上安装了中文桌面，个人感觉并不好，记录一下踩过的坑。

### 🀄 中文桌面问题

首先命令行桌面显示汉字后，命令行切换中英文非常困难，想索引到桌面并不容易。其次 Linux 下的中文非常难配置，就算配置好了也会在莫名其妙的地方受限，比如切换中文输入法、命令行切换中文等等问题。

所以就算你是电脑小白或 Linux 小白，不要害怕英文桌面，能用就用。中文只是桌面加了一层汉化，大部分时候你创建文件肯定还是创建英文为主。

### 📦 软件下载与镜像源

在 Linux 下下载东西非常困难，因为国内访问国外源下载和升级都非常慢，这时候需要镜像源。但用过的人都知道镜像源非常难搞，命令巨长无比。

给同样困扰的人提供一个小技巧：在自己常用的设备（Windows）上用 AI 做一个局域网传输工具，通过它将镜像文件快速传输到 Linux 下，软件就能快速转移过去了。

### 🌐 Verge 代理配置（重点）

Linux 下配置代理花费了我三四个小时，记录下正确做法：

1. **关闭系统代理，开启端口**：千万不要在 Linux 开启 Verge 的系统代理，Linux 似乎做了分离，它并不能代理浏览器，会直接导致什么都访问不了
2. **让 Linux 监听端口**：这一步需要自己学习什么是端口和监听功能
3. **浏览器配置代理**：在浏览器中配置 HTTP 和 HTTPS 访问端口，这样就可以实现浏览器通过 Verge 访问了

### 💡 软件操作建议

Linux 软件和 Windows 完全不同，所有软件不会像 Windows 一样把后缀直接显示出来，你得自己去看哪个文件是可执行文件。

如果你是对 Windows 的小白，到 Linux 下一定非常吃力。一定要提前学习：软件的可执行文件放在哪里、如何更好地操作软件等基础知识。
