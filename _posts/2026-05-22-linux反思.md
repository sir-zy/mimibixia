## 💥 长时间不关机的教训

在最近的学习中发现一个很重要的问题：Linux 并不是完全可以不关机的。我之前一直以为 Linux 系统会更加稳定，电脑可以不关机，结果导致长时间不关机后分区错乱，自己的系统搞崩了。

看起来 Linux 长时间不用也会出大问题，不过通过重新刷系统已经解决了这个问题。

这个错误告诉我们不能太理所当然，很多时候一些问题在你最开始就出现了错误而你并没有深入思考，结果造成了更大的错误。还好这个问题我发现的早并不是很严重，之后就需要好好对待电脑了。

## 🪟 Linux 与 Windows 的对比

同时最近也学习了一些 Linux 的管理工具。

Linux 没有 Windows 强大的自适应能力——Windows 更像一个强大的管家，帮助你去管理你的电脑使用，同时有强大的自我修复机制。而 Linux 更像没有管家的房子，你自己去完成各项任务：配置、学习等。

学习 Linux 真的是在强制地去学习电脑的知识，我也感受到了 Linux 的强大之处。
